保護耳朵

音樂人要好好的保護自己的耳朵。

現在隨身聆聽設備很方便，現在智慧型手機，都可以播放音樂，使的不管在路上、還是辦公室裡，很常會看到大家帶著耳機聽著自己的音樂。樂手是真的需要大量聆聽的，一方面吸收新資訊，一方面聆聽好的作品，對於提升音樂的素養是非常重要的。因為『 CD是最好的老師』… 不過 …

就因為現在聽音樂太方便了，無時無刻，不管什麼地點幾乎都可以聽，都可以沉溺在屬於自己的旋律世界裡 … 漸漸的這樣現象就開始出來了：

  1. 音量不自主的越開越大聲，有些大到我距離兩個人的位置都知道他在聽啥歌。
  2. 搭乘大眾交通系統時，不知不覺的越開越大聲
  3. 聽力不知不覺中漸漸受到傷害
  4. 樂手在練團室，如果又不懂得控制 punch，音量，越推越大…
  5. 如此，惡性循環…

年紀小可能還沒關係，但是長時間累積下來對於耳朵的傷害是無形的，甚至是永久性的傷害。

我曾經為了作音樂，長時間戴耳機，或者持續在音壓很大的地方作音樂。後來漸漸感覺耳朵有刺痛感，對於"音量"靈敏度降地很多。當我意識到的這事態的嚴重之後，我開始自律，盡量不戴耳機，除非必要，否則音量一定要限制。特別在坐公車、捷運等公共場所，這種地方通常都很吵，機械運作的音量就很大，戴耳機聽音樂的時候，很容易不自主的樂開越大聲，在這樣的環境就要越特別去控制音量。

吉他手很多時候，為了音色 (Tone) 的課題，需要把音量開很大。但以現代的技術來看，已經不像以前那樣。以前為了要取得自然破音(Overdrive)，需要真的把前級音量推到炸掉，然後得到吉他手最想要的破音音色。

現在數位科技發達，很多產品可以不用開很大聲就可以作出對應的效果。所以吉他手要專注的，應該是如何用耳朵判斷出音色的動態、特性等，掌握其特性之後，學習如何善用現有的設備，像EQ / Comp / Limit / Reverb / Delay / Gate / Modulation…等，還有最重要的就是吉他手本真的演奏技術，達到所需要的音色效果。

但是這一切都要靠一對能夠精準判斷的耳朵，所以把耳朵 “養好”，是非常重要的。

我去一些LiveHouse聽音樂時，發現都有一個特性就是，通常低頻都很多，猜測可能有幾個因素：

  1. 空間和輸出設備功率沒有匹配
  2. 吸音不好，散射太多
  3. 樂手是 Rock / Metal 掛的
  4. PA 大哥、樂手長期在那樣的工作環境之下，聽力已經受傷了，所以要開很大的音量才聽的到

如果是最後一點那就真的有點注意了。畢竟我們不是貝多芬，只剩一隻耳朵還能寫出曠世鉅作。要是樂手耳朵受傷了，不要說彈琴，可以能連平常要聆聽自己喜歡的音樂都會有問題 … 玩Rock / Metal的吉他手和鼓手通常也是耳朵傷害的高危險群。

如果不可避免的要長時間戴耳機聽音樂，我的方式是這樣的：自從那次的經驗之後，我開始試著把音量越開越小聲，讓耳朵即使在很微量的音量中，或者在很吵雜的環境當中，還能夠清楚的聽到想要的音樂。長期這樣"訓練"下來，因為耳朵能夠聽到高動態範圍，對於音樂製作的混音來講，對於Comp / EQ的應用，應該是有幫助的。

在練團室的音量控制，就複雜的多。比較簡單的方式就是戴耳塞，作基本的保護。然後選擇空間大一點，吸音好一點的練團室。有心的話，加強樂手們的對聲音基本概念和認知、認識聲音的物理特性，還有Comp / EQ的應用與概念。如果有固定的練團室，大家可一起嘗試調整、實驗、找到最佳的音量數據、人員位置、FX設定…等，而不是一直把音量一直堆大，如此只會惡性循環。
