閒聊 (價值觀) - 禮俗/實用價值

June 16, 2023 · 3 min read
Rick Hwang
GTCafe Studio

三節企業都會送一些東西給員工，像是三節獎金，有時候會多個禮盒，像是中秋禮盒。同樣的是喜宴，都會送看起很精緻的禮盒。或者一些社交活動場合，主辦單位也會送一些紀念品，像是 … 我想不起來。

我是個務實主義者，基本上物質就希望能物盡其用，定期清理出清沒在用的、不需要的，定期清理就是個慣性。東西的使用頻率，決定去、留。

※

所以每次收到包裝很漂亮的禮盒，不管中秋禮盒、端午禮盒、社群活動送的東西 … 吃的就是吃掉，但剩下的盒子，真的丟覺得浪費，留下來覺得多餘。很多人一定會說，那個盒子可以拿來收納，或者袋子買東西時可以裝。然後我回頭看那些依照這個想法，留下來的袋子、盒子，每次要用的時候，我從來沒有一次會想到他們。所以代表，有沒有留下來對我來說根本就沒差。當需要的時候，想不到的，對我來講就沒意義。

※

依照這原則，我也不太會有「蒐集」的特性，像是蒐集吉他模型、薩爾達周邊、FF 週邊、回到未來周邊 …. 都很喜歡，但從來沒有去買過模型或者公仔。同樣是買回來只有佔空間，然後沒有實用性。

房子最寶貴的就是空間，所以沒有實用性的，大多都不會留太久。不管他有多漂亮、或者 … 包裝看起來多高級。

※

除了理性的角度，感性的角度還是有，畢竟還是有血有淚的人XD

有紀念意義的，且是自己創造的，一定是留下來，像是寫下來的筆記、樂譜，很喜歡的小說、書籍。買來的，像樂器 / 吉他，因為長時間使用，基本上也都是有意義的。

但是另外一種雖然有紀念意義，但卻是尷尬，最後會丟的。最近丟的是，過去幾年參加研討會時，主辦單位給的識別證帶，實在是太多了，每次都送一條，每條都很精美，但都只用一次。。。所以時間到了，就丟。

還有丟的是一些也是研討會的東西：T-Shirt。T-Shirt 算是有實用性的東西，但是，應該很少人看我穿過那些衣服，除了 Java / AWS 這兩件，我不太喜歡被隨便貼標籤，所以有特定標籤的服飾，大多都不會穿，會穿的也都是素為主，基本上衣著以乾淨，簡單為原則。所以有特定標籤的，幾乎都不會穿，所以很多連拆封都沒有，全部打包，捐出去了 …

※

今天又收到一個很漂亮的禮盒，然後，東西看起來很好吃，禮盒看起來很高級。下意識：

這垃圾分類要怎麼分啊。。。好麻煩 QQ

以後應該會被活動列為拒絕往來戶 XD

※

還好現在不太會收到朋友送的東西 不然可能會有點尷尬
