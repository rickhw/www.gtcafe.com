記憶形成的四個步驟

June 22, 2024 · One min read
Rick Hwang
GTCafe Studio
  1. 編碼 Encoding: 將外來的資訊 (感官/情緒), 轉化成神經語言
  2. 固化 (Consolidation): #海馬迴 (Hippocampus)
  3. 儲存 (Storage)
  4. 提取 (Retrieval)

如果四個步驟沒有完成，當下的經驗，就不會變成記憶。像是 Context Switch、或者根本就不在意的任務。

  • 核心記憶決定你是誰
  • 人為什麼要遺忘？ BingBong 為什麼得死？
  • 為什麼長大後，會忘掉曾經很喜歡的事物？就像是曾經很喜歡的玩具、故事、朋友、老歌。

如果劇情主角改成男生 青春期應該會很好笑XD
