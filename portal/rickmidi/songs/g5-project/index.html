G5 Project

介紹日本吉他演奏專輯

介紹幾張我很喜歡的專輯。收錄五位日本的 Session Guitar Players 各自的音樂，無論在作曲、編曲、演奏都相當優異，名為 G5 Project 的專輯。

Words by a2c

首先介紹的是 G5 Project - 2007 專輯裡的 “Words”，非常日式流行的編曲風格，整首似乎就是可以隨時配上歌詞，搭上日劇主題曲或者插曲。特別是的是這首曲子的副歌，用人聲吟唱作為主旋律的襯底與和聲，更加加強了戲劇性與故事性，每次聽我腦袋都跑過很多畫面。

Gratitude by a2c

和 “Words” 一樣都是代號 a2c 的作品，另一首很多人都喜歡的曲子：“Gratitude”，收錄在 G5 Project 2010。同樣的是強烈的個人風格與編曲技巧，聽到歌曲間奏的人聲吟唱，就知道是誰的作品。

從看到曲名 Gratitude (感激)，前奏和第一段用 尼龍吉他詮釋主題，編曲時，用尼龍吉他搭配 bell (鈴聲背景音色) 音色來表示回憶、記憶、回顧、過往、思念、惆悵 … 等意境。

間奏 full band 將過往拉回現實。第二段用電吉他重新詮釋主題，回到的現實衝擊與心境。副歌同樣用人聲吟唱做和聲。

Bridge 則展現了過往與現況的衝擊，最後收回主題，回到現在：Gratitude。

Over the Bridge by Takajii

“Over the Bridge” 是一首爽歌。G5 成員中的 Takajii 作曲、演奏，技巧非常的高。不過我最喜歡最後一段 Bridge 之後吉他和鋼琴的輪流 Solo。

這段是 Takajii 本人的 Bridge 演奏。

Rain by Takajii

提到 G5 的成員 “Takajii” 不得不提他另一首很有名的曲子: Rain。旋律性非常好的抒情演奏曲，同樣的非常有畫面，加上 Takajii 爆發式的演奏技巧，情緒掌握得很好。

小聲的說，這首我也練過。。。不過練不出他的張力與味道 = =

影片也是 Takajii 本人的詮釋。

Fragile Seasons by Takajii

G5 2013 年的專輯，同樣的 Takajii 的作品: “Fragile Seasons” 同樣是很有畫面的旋律與編曲，跟主題非常搭，很適合現在的季節。

演奏用很多搖桿技巧。

影片是別人 Cover 的，但詮釋得不輸給原曲。

Break Down by Takajii

前面介紹的，大多都是慢歌。來首搖滾的，收錄在 G5 Project 2005 (十年前@@) 專輯的第一首，同樣是 Takajii 的作品。簡單說就是爽歌～～

其實我就是喜歡 Takajii 的風格 XD

Journey by godspeed

Journey 這首曲子是 godspeed 的作品。聽這名字就覺得很嗆，居然有人的名字叫做 “上帝的速度“。很多吉他手都很迷戀技術，特別是速度，網路上一堆世界紀錄，我猜這位吉他手也應該曾經是這樣吧？

註：有網友說 “godspeed” 不是上帝的速度，他本人說 “一路平安” 的意思。

“godspeed” 其實是 “G5 Project” 的發起人。不過我只喜歡他這首曲子，不是因為技術很炫，而是歌曲的意境很深遠。歌曲用進行曲作編曲，有別於樂團形式，管弦樂團做主要的協奏，低音部為銅管樂器，弦樂的編曲可以感覺出他想營造出史詩的氣勢，有別於一般電吉他演奏曲的編曲。

歌曲的曲式安排和他的音色，算是很搭。聽過不少電吉他和管弦樂團的搭配，有時候覺得只是送作堆的做法，音色相互吃 tone 很嚴重，特別是電吉他的 DS 和小提琴放在一起，會很容易互相『蓋台』。

影片是 “godspeed” 本人的詮釋。
