現實與理想的掙扎

整理文章的同時，看到很多年的日記 …..

音樂教室因為老闆記錯時間，所以晚了一個小時才開門。外面下著大雨，幾個學生跟老師在教室門口等，聊天。一個老師說到，現在台北能作的場子越來越少，待遇也越來越不好。但是爭著要作的人卻越來越多。

一場演出從過去近萬的行情，到現在一場大概 2500，甚至更少，樂手們的待遇是一年不如一年。現實面來說，一場 2500 的表演，在台北能夠生活，一個月少說要接個十場，有時候還要到音樂教室兼一些課，才會有一個基本穩定的收入。如果有唱片公司找上錄音，那更是可喻不可求。另外其他的路線就是走所謂創作，賣作品給唱片公司.

其實樂手在台灣這樣的環境能生存下來的，每個真的都不是省油的燈。一邊是夢想，一邊是現實。在這之中，很多人往往在取捨之間，就被扼殺了....放棄的放棄，轉行的轉行 … 不久前，我才遇到一個轉行賣車的鍵盤手 …

一個老師說，最近有一個錄音室的樂手，因為負債，加上出車禍，理賠 + 手受傷 (廢了)，人生幾乎是毀了 … 現實就是這麼的殘酷。當然，有些人就成功的轉型了，從樂手的身分變成所謂的 "藝人" (幕後到幕前)。黃品源、伍佰就是很成功藝人。當然，他們還是不錯的吉他手。

這樣的環境，很現實，很殘酷，很多人在裡面，想逃卻逃不出，想退出短期間卻也沒辦法....但是卻也很多人想跳進去，像我 … 其實我也不是很想真的跳進去，只是想沾一邊吧 … 就像吃生魚片，只會想沾一點點芥末醬 … 但是不會想反客為主 …

我還是個為生活努力且單純的上班族，每天乖乖的去上班，下班 … 為家人，為生活努力 … 只是週末有空，有機會，有舞台，就去表演，職業的水準，業餘的身分 … 或者有一些好朋友，週末大家一起聽音樂，一起玩音樂 …

人生是一場現實與理想，取捨之間，矛盾的掙扎，今天放棄也許明天就會後悔 … 但是有的時候，並不是真想放棄，只是累了...這種的情緒，這樣的心情，常常在生活之中反覆 … 有時候我會停下腳步，試著想想初衷，想想當初彈 C 和絃 + T1213121 的感動 (彈到流口水的感動)，然後，再繼續走下去 …

幾年下來，還好我沒有被環境扼殺掉對音樂的喜歡，也沒有賣掉任何樂器。很幸運的，在工作之餘，有機會從事吉他教學；然後又因為跑了一些場子、組了一些團，認識了一些好朋友。

雖然有的時候也會想 “音樂可不可以當飯吃？” 這種問題，不過，還好這問題很多年前我就有答案了。我不會羨慕舞台上那些 Guitar Hero、或者 Super Star，因為那不是我的人生，也不是我想要的生活。
