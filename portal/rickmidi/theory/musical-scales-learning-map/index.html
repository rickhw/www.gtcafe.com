音階學習地圖

發布日期：2008-10-04

下圖是目前常見的音樂音階學習地圖，圖表的底色分別表示：

  • 深藍色：基礎、最重要的音階
  • 淡黃色：進階的調式、藍調部分
  • 黃色：屬於民俗調式系統
  • 淡藍色：屬於進階音階
  • 淡褐色：屬於 Jazz 範圍的調式音階

學習建議次序為：

  1. 深藍色
  2. 淡黃色/黃色
  3. 淡藍色
  4. 淡褐色

圖表是學習的地圖，音階關係請參閱音階基礎

※版權所有，未經允許，請勿隨意轉載※

這些音階不是全部都要學的，依照不同曲風需求，會使用不同的音階。但是一般的學習來看，深藍色的基本概念是要熟悉的。具備基本的大小調、五聲音階，再搭配對映的和聲概念技巧，其實就已經可以滿足大部分音樂的需求。

如果基本的音階還無法滿足，可以開始嘗試調式的概念，音樂更有張力、更有獨特風味。這類的音樂像是藍調、拉丁、放克、爵士、金屬曲風等。

接下來就是屬於進階的小音階系統，主要包含了和聲、旋律、爵士小調。前兩者喜歡重金屬的人會很常聽到，後者爵士小調則是學習爵士樂的 “爆點” 。

爵士學習者從小音階系統的各調式開始，會有三大類，最後還有對稱系統等更進階的。這些東西比較不常出現在一般的流行、搖滾、藍調、金屬 … 出現也是點綴的成分居多，但這些卻是 Jazzkid 的入門基本功課。要熟記這些東西不難，但是要能夠靈活運用他們就難了 …

音階這麼多，怎麼學？簡單說，穩紮穩打才是關鍵。千萬不要跳著學習，因為這些音階大部分都有前後關係，在耳朵還沒有辦法分辨出來基本的音階聲響、基本合聲時，跳著學習其他進階的音階，可能會讓腦袋過載、更加的困惑，不知所以然。

所謂『耳朵可以分辨出音階聲響』，簡單說，就是假設已知一首大調曲子進行中，突然出現一個大調音階以外的音，或者突然轉調時，耳朵可以知道音階已經發生變化。

或者唱 KTV 時，朋友 Key 沒對到 (這大部分的人都可以察覺)、走音可以馬上聽出來 … 等。

具備類似的基本聽力，再去學習其他的音階，是比較適合的。
