次屬和弦 (Secondary Dominant)

發布日期：2009-04-01

基本概念

次屬和弦是一種 和聲技巧，和 調式內轉 (Modal Interchange)、TSD 和聲代換，都算是同類型的東西，他們的目的都是要製造出與原本歌曲和聲不同的感覺、張力，但是又不會離原曲太遠 (Outside)。

在說明次屬和弦之前，我們先來釐清一些基本概念，也就是什麼是 屬和弦 (Dominant Chord)？

屬音 (Dominant)

Dominant 這個單字是個形容詞，有 支配的、控制的、統治的、佔有強勢的、優勢的的意思，可以等同於 Controlling。它的動詞是 Dominate，就是 在 ... 佔有重要地位 的意思。所以套用在自然音階，可以想像成七個音符裡，第五個音是佔有重要位置的音，中文把它翻譯成『屬音』。

主要屬和弦 (Primary Dominant)

首先從字面上來說明 “Secondary Dominant”，“Secondary” 字面上就是 第二的、次要的、輔助的，簡單說就是老二，他的相對形容詞就是 主要的、首要的、第一的 (Major、Primary、Premier)。所以有 Secondary Dominant 當然就會有 Primary Dominant。

大部分 Key Center 的歌曲當中，其順階七和弦裡，都會包含了一個主要的屬七和弦 (Dominant 7th)，自然大、小調都是五級 (自然小調有時候就會把五級的小七改成屬七，詳細解釋請參閱 “小調音階” 的說明)，這個屬七和弦一般用來判斷歌曲的終止，或者決定歌曲的調性。

重點：用來判斷 歌曲終止 或者 歌曲調性 的屬七和弦可以稱為 主要屬和弦 (Primary Dominant)。

在大部分 Key Center 歌曲中，一個調裡，只會出現一個屬和弦。這個屬和弦代表著調的主要屬和弦。如果出現第二個呢？那可能就是所謂的次屬和弦。

次屬和弦 (Secondary Dominant)

Secondary Dominant 我習慣翻成 次屬和弦，有些書翻成二階屬和弦、次要屬和弦、裝飾屬和弦、附屬七和弦、應用屬和弦 … 他們的意思都是一樣的。原文有時候也會寫成 Applied Dominant。

在主要屬和弦裡已經提到的基本概念：在一個調裡，僅會出現一個屬和弦，如果出現了兩個屬七和弦，我們就要判斷那一個是主要的，哪一個是次要的，例如著名的搖滾經典加州大旅館 (Hotel California)：

整首歌，頭尾來看，可以得知是 B 小調，對應的主要屬和弦是 F#7。歌曲當中出現了另一個屬七和弦 E7，我們把它看成是某個調的屬七，剛好 E7 前面是 A，兩者剛好是 I - Ⅴ 的關係，所以 E7 我把它解釋成歌曲第一組 次屬和弦。

另外還有一組可以解釋成次屬和弦的是第四、五小節，G - D 的關係也是 I - Ⅴ，只是這邊的 D 不是屬七，但是眼熟的人應該可以感覺出來 D 是 G 的五級。所以我把它解釋成第二組次屬和弦。

這個和弦進行裡，有兩個屬七和弦，但是有三組屬和弦。從調性判斷哪個是主要的，剩下的都是次要的。

次屬和弦一般來說，經常發生在：大調的 Ⅱ、Ⅲ、Ⅳ、Ⅴ、Ⅵ。 I 是因為 I 的 Ⅴ 級就是調性的主要屬和弦。

級數 I Ⅱ Ⅲ Ⅳ Ⅴ Ⅵ Ⅶ
主和弦 CM7 Dm7 Em7 FM7 G7 Am7 Bm7-5
屬和弦 G7 A7 B7 C7 D7 E7 F#7
表示法 Ⅴ/I Ⅴ/Ⅱ Ⅴ/Ⅲ Ⅴ/Ⅳ Ⅴ/Ⅴ Ⅴ/Ⅵ Ⅴ/Ⅶ

一些常見的例子：

| C / / / | E7 / / / | Am / / / | G7 / / / |
| G / / / | C / / / | A7 / / / | D7 / / / |
| C / / / | C7 / / / | F / / / | G / / / |

次屬和弦與減七和弦

在 “減七和弦” 一文中提及過，減七和弦最常用在和弦的銜接，例如 1 - 2m、4 - 5 的經過和弦。

像是：

| C / / C#dim7 | Dm7 / / / |

或是

| F / / F#dim7 | G7 / / / |

這種用法，可以用次屬和弦來聯想，像：

| F / / D7/F# | G7 / / / |

把 D7 看成 G 的五級，其中：

D7/F# (D F# A C) 和 F#dim7 (F# A C Eb)

聲響很接近，以可以互換。一方面有有屬和弦效果，又讓 bass line 形成半音進行 (F - F# - G)，是很實用的老梗。

另外，F#dim7 跟 Am6 是同音異名的和弦 (詳見 “關係和弦?” 的說明)，所以這段和弦進行又能被改成：

| F / / F#dim7 | G7 / / / |
| F / / D7/F# | G7 / / / |
| F / / Am6 | G7 / / / |

所以導入一些類似的概念，可以發現很多有趣的現象。

結論

次屬和弦 和 調式內轉 (Modal Interchange) 很容易搞混，但是就實用上來說，我個人覺得只要能夠抓住調性和聲的基本概念，也就是本文中第一段提及的觀念，那麼這兩種和聲技巧，就不會搞混了。

另外，藍調的和聲不要用次屬和弦來解釋，基本上他天生就長那樣。
