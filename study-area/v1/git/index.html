branch operation

Branch in local

# list the branch in local
$ git branch -v

  draft/<branch_name_1> 2950234f update
* draft/<branch_name_2> 332b8a0e update
  master               0e0d1734 update on

# 更改 branch 名稱
$ git branch -m <old_name> <new_name>

# 刪除 branch
$ git branch -d <branch_name>
$ git branch --delete <branch_name>

## delete in force
$ git branch -D <branch_name>
$ git branch --delete --force <branch_name>

# merge from <branch>
$ git merge <from_branch_name>

Branch in remote

# list the branch in remote
$ git branch -a

  draft/<branch_name_1>
* draft/<branch_name_2>
  master
  remotes/origin/1.0.0
  remotes/origin/HEAD -> origin/master
  remotes/origin/main
  remotes/origin/master
  remotes/origin/draft/<branch_name_1>
  remotes/origin/draft/<branch_name_2>

# remove branch in remote
$ git push --delete origin <branch_name>
$ git push -d origin <branch_name>

To <vsc_host>:<username>/<repos_name>.git
 - [deleted]           <branch_name>
