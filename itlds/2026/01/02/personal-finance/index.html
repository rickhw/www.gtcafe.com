聊聊個人理財

#大人學 經常提到一句話：「用經營公司的思維，經營人生」

經營公司經常提到 #產銷人發財 這五大核心概念，也就是生產、行銷、人事、研發、財務，個人部分大部分沒這麼複雜，但可以用這個思路框架來發展自己的 #年度計劃，大概如下：

  • #產 (生產力、Productivity): 生產力的源頭是健康的身體，有健康的身體才能功高品質 (Quality) 的生產力，像是良好的運動習慣、飲食、睡眠 ... 等。
  • #銷（行銷/業務, Marketing）：個人品牌與影響力，你是誰？怎麼讓別人找到你？別人會怎麼形容你？你的一小時有多少價值？
  • #人（HR 人力資源): 我把這概念對應到人際網絡，像是哪些人是合作夥伴？哪些人是導師、Role Model、情報中心 ... 人際網路的強弱連連結、有毒的關係 ... 等。
  • #發 (R&D): 自我成長，每年學會的跨領域新技能、閱讀、動手？是否是為自己的未來佈局新能力？複利自己？
  • #財 (財務、投資、FinOps) : 現金流、開源 (投資)、節流、預算、複利

其實大部分的人都沒有像公司那麼複雜，但有幾個項目，我是每年都會整理的，像是預算。

預算​

工作上，因為職務關係，每年大概都要編列預算，控制支出。編列過全公司的 IT 預算、公有雲預算、或者單位系統預算 ... 等不同範圍，也處理過像是 雙十一支出這種特殊任務的預算 (YoY, Year on Year)。

IT 預算就是全公司的軟體費用、新員工硬體採購、軟體訂閱、公有雲費用、採購合約 ...等。有一陣子在做公有雲的結費計劃，包含 AWS / GCP / Azure 的預算與節費計劃，所以定了一套管理機制來圈選，其實就是利用 AWS Tag Policy 的方式，然後從組織、產品、系統三個維度去圈選，然後每個月才有辦法產出 MoM (Month on Month) 報表，讓各單位去往下的結費計劃。我的書提到的三體思維其實是從這裡開始的。

那時候沒有 #FinOps 這個詞，後來這個詞廣泛流行後，我才知道我已經做了幾年了。背後核心概念就是控制預算，知道錢花在哪裡，往前推算就必須要能夠圈選、分類，才能夠分析，所以產生了 Tag Policy。Tag Policy 訂好之後，就是圈的動作，讓團隊成員去執行。過程就不限制方法或者技術，總之，從小範圍開始，像是測試環境的資料圈選，每個月出報表給各單位看，by Org / by Product / Project / by Market ... 等各種維度。要能量測，才能改善。後來把範圍從測試環境資源，拓展到 Production，雖然更複雜，但讓大家更有感錢花在哪裡。再配合老闆給的目標，例如該年度的支出費用跟去年一樣 or 降低去年的 10%，大家對於自己產品在要 IT 資源時會更有感。

在工作上習慣做這樣的事情，其實不是第一次，更早的是在當兵的時候了。

預財士​

當兵時我是單位的行政預財士，業務之一是負責單位的財務規劃，包含採買、預算編列、會計結算等業務。單位雖小，但五臟俱全，也因此學到一些財會的概念。

那時候每個月都要採買單位的伙食，所以每個月要花多少錢採買，預算是要算得很清楚的。隔月結算的時候也都要驗算與結算的，所以很習慣要為每週開一個預算表，三餐吃什麼，大概花多少錢，然後這個月最後一週，就把下一個月 30 天的伙食都開出來，大概要花多少錢，然後報上去。伙食只是其中一項而已，當兵還有專業技能的帳要報，國防機密就不說了。

會寫程式的好處，就是這種事情都交給電腦計算，我還沒去單位的時候，負責的學長每個月結算最後一週是要大家不要去打擾他，然後就看他拿著計算機跟紙本在那裡算。因為軍中帳算錯是會被抓去關的，好像是刑法，因為會被當作是貪污，是很重的刑責。後來這個業務就變成我的工作之一，只是我就寫程式處理了，包含伙食採買的部分，從採買預算、到其他軍中採買業務，都直接用程式搞定，然後我每個月月底一樣在辦公室裝忙，其實一秒就算完了 XDD

個人財務​

處理過公司那麼複雜的「資產」、「預算」的經驗，回到個人財務，真的就是小巫見大巫，個人財務再複雜，都沒有一家公司的複雜，可能連千分之一都沒有。

最大的差異，帳面上的數字都是自己的錢 XDD

最大支出​

出社會工作之後，我一直都有在記帳，每一筆支出，都有記錄。

不久前跟朋友聊到，我其實每個月的支出大概都在 2 萬台幣以內，我個人最大的支出大概就兩個項目：1) 電腦; 2) 樂器

電腦是個人的專業工具，吃飯的傢伙，是一種投資。因為我把這項當作投資，投資我自己學習的環境，讓我自己可以有更好的資源可以不斷地提升自己。所以工作後我買的電腦都差不多是當時的頂規。工作後的第一年春節 (2005)，我買了第二台 Notebook - IBM ThinkPad T42P，那時候花了 67K，而那時候我的月薪只有 34K。之後平均每 3 ~ 4 年會做一次電腦升級，每次都是當時期的頂規。我常講的例子是我在 2012 年去光華商場組的 i7 桌機，那時候的規格是 i7-950 / 24GB / 2TB，這規格放到今天，還是屌打一堆。後來開始用 Mac，從 iMac 的 2015LATE 27' i5 / 32GB / 2TB、到 2019LATE 27' i9 / 64GB / 2TB，到最近一次採購是 2022 年的 Macbook Pro M1 Max / 64G / 2TB，都是頂規。

電腦是個人投資，也是生財器具，樂器就不是了，純粹是個人的興趣，所以我不會買最好的琴，因為彈的好不好聽，還是要看投入多少時間練習才有意義。工作之後，買過不少樂器，算一算，電吉他 5 把、Keyboard 3 台、周邊 (錄音介面、監聽、軟體)、吉他效果器、真空管音箱 ... 一堆 ... 最貴的吉他 40K、最貴的 Keyboard 90K。。。算起來也蠻恐怖的 XD

不過樂器設備大多可以用很久，現在手上的吉他三把都是二手的，最近買的是 2021 / 2022 分別買的 Fender & Gibson，彈了二十多年，總算是圓了夢。但這兩把個別是 38K / 40K，相對於全新的 80K / 100K 其實是少很多的。

吉他跟電腦不一樣，保養的好，是會保值的。

記帳​

記帳的習慣，是從當兵就開始了，那時候當 #行政預財士，學到如何 #會計結算，是最有幫助的技能。所以當兵時那少少的薪資，就是我記帳的開始，從二兵 (5.7K) 到升下士 (14K?) 一路存，除了存錢，也計算退伍後的生存問題。

到台北工作後，每天都在記帳，用過很多記帳軟體，也自己寫過。工作的前十幾年，用的都是單機版的記帳軟體，每天下班第一件事就是記帳，每個月月底就是看存了多少錢。

後來用 iMac、iPhone，就換手機 / 電腦同步，叫 MoneyWiz，訂閱制，也用了十幾年。

這幾年，則改用 Google Spreadsheet，簡化整個流程，不再計算很瑣碎的細節，而是直接用收入、支出、投資三個角度計算。算是第三階段的進化。

個人預算 與 獎勵政策​

工作需要編列預算，個人我也一樣有固定的預算再算，例如：每季的學習預算是 5000，通常我都是書買好買滿，以前還常常會超支。

預算是每年年底、年初要規劃的，包含一些大型支出、生活支出、所得稅、學習預算、預備金 ... 等。

除了基本的生活支出，前面提到關於電腦、樂器的預算，每年都會編列，但是是累積制的。也就是大部分都不會用，會累積幾年一次用掉，而且是要附帶條件，像是完成什麼特定任務、或者學習了什麼技能。

樂器就是要練好什麼歌，或者完成一些核心的創作 ... 等。

這樣的方法是提醒自己，有預算不見就可以用，而是真的要提升能力的，再來擁有那些設備才有意義。讓設備加值自己，不是買了設備來炫耀自己有什麼。

以前教吉他的時候，上課有時候學生會說：「老師你的琴比較好，所以彈起來才能夠「趁趁趁」 (電吉他的 Power Chord 狀聲詞)。」

這時候，我的反應就是：「好，那我們來教換吉他上課。」然後就這樣到下課，我就問他：「你現在覺得是吉他的關系？」結論都是：功夫不到的問題。

我的 Blog 有寫過這段話：

  • 不練琴的吉他手 + 十萬的琴 + 三十萬的設備 = 很爛的聲音
  • 有在練琴的吉他手 + 5000 的琴 + 5000 的設備 = 還不錯的聲音
  • 大神級的吉他手 + 5000 的琴 + 5000 的設備 = 超讚的的聲音

所以，時時提醒自己，別只是想買設備，但都沒在練琴。所以如果我已經下手了，代表 $$ 準備好了、歌也練好了。

由儉入奢易由奢入儉難​

小時候 (應該是小二?) 有一段時間，很喜歡自己做 #立體模型，就是拿西卡紙，然後畫 3D 透視圖，剪裁後，黏成模型。那時候很常做的是船、車子。(不要問我為啥會自己畫 3D 透視圖，就是會 XD

那時候經常要找材料，最常用的是包裝盒的紙、厚紙板，後來美勞課知道有一種叫做西卡紙的，就叫老爸買給我。

老爸也真的買了給我剪裁，但發現我都很浪費，一張四分之一開的西卡紙，就從中間剪裁，浪費掉很多紙。記憶很深刻的是，老爸有給我機會教育，說這一張紙價值 5 塊，你只在中間挖一塊減掉，剩下沒用到的有 1/2 大小，這樣是不是就浪費兩塊多？這次的機會教育，讓我知道要好好善用每一份資源。

後來這件事情就好像種在我的基因裡，生活中不管啥東西，都會習慣性地讓他發會最大的利用價值。像是一抽的衛生紙，可能只是要擦一些小東西，就習慣性的紙撕下 1/5 的大小使用，剩下的放在一個 temp 地方，下次就會去那裡找，又私下 1/5 大小用。。。不過有時候也會忘了已經有 4/5 放在 temp，然後又抽了一張，又撕下 1/5 使用。。。temp 那區域就一堆 4/5 的衛生紙 .... XDD

水、報紙，也是這樣。墊桌面的報紙，會反覆用好幾次，正面用完，用反面，反面用完用裡面，再翻面。。。(好像有點髒 XD

電費我自已因為是電機畢業的，所以房子的電費是全局控制的，每個設備消耗的電力，都有在試算。如果有浪費的，就是算過可以接受的，像是一些 LED 燈。

工作久了，有了一些積蓄，其實可以承擔的費用是比以前好很多，但是小時候老爸的提醒，已經烙印了。雖然會買很貴的電腦或樂器，但還是有設定一套規則，讓投資或者享受可以更有價值。

年度預算​

每年年底 會看今年每月支出，平均支出、重要支出，非預期支出 ... 等項目。Review 有沒有好依照預算執行，非預期支出處理得如何？

從這些 Review 過程，可以瞭解到，生活其實可以很簡單，可以有多少餘裕，每年燒一次腦，回顧過去，規劃未來，有餘裕的財務，生活才更知足也常樂啊。

用公司財務預算的思維，經營自己理財與人生～

PS2: 我很常聽大人學的 Podcast，但我沒上過他們的課，所以應該不會有雷同。
