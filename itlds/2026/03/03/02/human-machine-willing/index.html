人類、機械、意識

前言​

我在 1/10 整理一篇「缸中之腦」的想法及整理相關藝術作品。最近我反覆思考幾個要素：「人類、機械、意識」，然後拆解為載體（硬體）與意識 (軟體) 註一，並探討它們之間的移轉 (Transfer) 與 複製 (Clone) 的種種定義與相關藝術作品。

整理到最後，我發現整個扣住一個最核心的哲學問題：「我」究竟是什麼？如果將大腦視為處理資料的硬體（CPU 或 GPU），而意識是運作其中的「軟體」，那麼人體的五感（視覺、聽覺、觸覺等）不過是接收外部訊號的 I/O 介面。循著這個基本邏輯，用我慣用的 #矩陣思維，逐一探討每種設定背後的意義、哲學觀點與代表作。

註一：我先不用靈魂，而是用意識。意識我認為是基於科學的概念，靈魂我當作是是另一個世界的想法。

矩陣思維的維度有：人體、異體的排列組合，搭配有無意識，排列組合分成三個大篇章，整理如下。

第一章：意識駐留，載體抽換（改變硬體）​

當意識與記憶沒有發生移轉或複製，而僅僅是改變了乘載的軀殼時，我們面臨的是「忒修斯之船」註二 的經典悖論：

當一個人的零件被逐漸替換，他還是原來的那個人嗎？
註二：《忒修斯之船》是一個古老的哲學思想實驗，探討「同一性」與本質的變化。源自古希臘，描述雅典人用新木板逐一替換忒修斯之船的舊木板，直到所有部件都更換後，該船是否還是原來的船。它引發關於事物本質是由「組成材料」還是「結構形式」決定的爭論。

▌純粹的血肉替換：人體 + 人體​

這是最貼近當代醫療現實的組合，也就是器官移植，像是心臟移植、眼角膜移植、腎臟移植、肝臟移植 ... 等外科手術，在現代醫學已經逐漸成熟。極端一點的還有腦袋換到新的身體的「人頭移植」，但實務上目前有在狗、老鼠身上實驗，但幾天之後受體就死了。

無論是器官移植或是換頭手術，它探討的是「身體與意識的同一性」。如果大腦是處理意識（電子訊號）的硬體，那換上別人的軀體，這些訊號的運作是否會受新軀體的生物反饋影響？在文學經典《科學怪人》中，拼湊的肉塊孕育了對造物主的悲憤；而在電影《逃出絕命鎮》裡，肉體的替換則淪為掠奪他人生命與青春的殘酷手段。

▌血肉與齒輪的交織：人體 + 異體 (Cyborg)​

異體指的是非生物結構，像是機械義肢、機械手臂等，可以為截肢病患，重新獲得新生命，像是換上新的腳，讓女孩可以重新跳舞，換上機械手讓男孩可以彈鋼琴 ... 等。這些異體仿生機械可以透過大腦的訊號，產生控制指令，進而做到一定程度的控制。但現在的異體還是屬於輔助功能，還不到強化人類的四肢或五感。

當人類選擇用機械四肢或感官強化義體來補強肉體時，我們進入了「超人類主義」的領域，這也是 CyberPunk (賽博龐克) 文化最核心的命題。這背後探討的是「生物性的流失比例」：當肉體不再提供飢餓與疲憊的生物反饋，意識這組電子訊號是否會產生變異而變得冰冷？這正是遊戲《Cyberpunk 2077》與動畫《邊緣行者》(Edgerunners) 中「神機錯亂」設定的底層邏輯。

異體除了人造物的機械結構之外，也可以是其他生物，人體 + 生物體，這已經進入恐怖故事、或者搞笑漫畫的瘋狂想像力了，日本經典漫畫《寄生獸》則是這樣的題材，但是他探討了自然、環保、生命的價值等更深層的意義。

▌最後的生物堡壘與 I/O 接口：人腦 + 異體（缸中之腦）​

伊隆馬斯克 (Elon Musk) 的腦機接口公司 Neuralink 目前正在做是透過一個介面，傳輸人類大腦的電訊號，透過這個介面解析之後，就可以讓人腦直接控制外部的設備。當 Neuralink 成熟到可以商用量產，人類大腦等同於裝上了 USB 接口，透過這個接口就可以控制外面很多東西，也可以取代五感接收外面的訊息。既然五感只是輸入大腦的訊號，超級電腦完全可以從外部餵給大腦逼真的電子訊號，同時也能將大腦指令傳給遠端的人形機器人。

最後，人類全身上下只剩下「大腦」這塊濕件 (Wetware) 是原裝的，這便是著名的「缸中之腦 (Brain in a Vat)」思想實驗的終極具現化。配合馬斯克的 Tesla Optimus (擎天柱) 機器人，人類就彷彿是「住」一個可以置換的機器人身上。

如果大腦接收的電訊號與真實世界完全一致，大腦根本無法判斷自己是身處於現實身體中，還是泡在營養液的缸子裡？電影《駭客任務》(Matrix) 便將人類身體當成電池囚禁，意識則作為數據流投射到名為「母體」的虛擬空間；而這概念的源頭，可追溯至 #菲利普狄克 (Philip K. Dick) 的《尤比克》(Ubik) 與 #威廉吉布森 (William Gibson) 的《神經喚術士》。日本知名動畫《攻殼機動隊》同樣是在這種極端設定下，探討被禁錮在鋼鐵牢籠中的有機大腦，該如何證明自己的意識依然屬於「人類」。

▌記憶竄改與真實的崩解​

當大腦有了 I/O 接口，隨之而來最可怕的應用便是「記憶的讀寫與竄改」。人的記憶本來就不可靠，有時甚至會像 AI 一樣因為上下文過長而產生「幻覺」與錯亂（如 #曼德拉效應）。如果記憶只是一種可以被覆寫的電子數據，那什麼才是真實？

《魔鬼總動員》(Total Recall) 中被植入完美假期記憶的男主角，直到最後都無法分辨夢境與現實；日本漫畫神作《攻殼機動隊》裡那位以為自己有妻女、為養家而辛勤工作的清運工，被駭客揭穿一切只是植入的虛擬記憶時，那一幕極度悲涼，完美詮釋了數據被偽造的恐懼；而英國單元劇《黑鏡》(Black Mirror) 則反向探討了當晶片記錄下一切細節、「遺忘」這個大腦的保護機制被強制剝奪後，人類意識將面臨的超載與崩潰。

▌矽基生命的覺醒：機器腦 (AI) + 機器體​

當完美的機械軀體配上高度發展的 AI 腦（例如搭載了頂級 LLM 的實體 Tesla Optimus），這是探討「意識湧現」的舞台。當一段程式碼複雜到能夠模擬情感、懂得恐懼死亡時，我們是否該承認這組電子訊號已經具備了等同於人類的意識？

知名電玩遊戲《底特律：變人》深刻描繪了仿生人打破程式限制、覺醒自我意識的掙扎；而電影《魔鬼終結者》與科幻大師艾西莫夫 (Asimov) 的小說《我，機器人》則展現了純粹理性下的機器意識；電影《駭客任務》(Matrix)前傳，則描述機器人亞當與夏娃到人類聯合國請願希望簽署和平協議，機器人與人類和平共存，但人類沒接受。

▌無形的神明或幽靈：機器腦 + 無實體載體​

褪去了實體軀殼，只存在於雲端數據流中的 AI，探討的是「非具象存在的孤獨與全能」。電影《雲端情人》中的作業系統 Samantha，展現了無肉體意識如何在訊號處理的高速進化後，因為運算頻率與人類徹底脫節而選擇離去；《2001太空漫遊》的 HAL 9000 則展示了純粹邏輯迴路在面對指令矛盾時的冷血瘋狂。

▌反向入侵的寄生：機器腦 (AI) + 人類軀體​

這是一種顛覆性的組合，探討「主控權的完全剝奪」。電影《升級》中，原本輔助人體的 AI 晶片最終接管了神經系統，奪取了肉體控制權；《駭客任務》中的史密斯探員，更是能將自己的代碼覆寫到母體內的人類肉身上，呈現了一種數位時代的訊號劫持。

第二章：意識的剪下、複製與貼上（訊號流動）​

當我們將「意識」視為一種可以讀寫的電子數據時，「自我」的獨特性便面臨了最嚴峻的挑戰。

▌連續性的幻象：意識移轉 (Transfer)​

「移轉」指的是將「意識」這組特有的電子訊號從 A 載體抽出，寫入 B 載體，且 A 變成空殼，不存在意識，不是植物人，而是一具沒有靈魂的軀體、或稱殭屍。從工程角度來看，數位資料無法真正的「移動」，只能「複製後，刪除原檔」。那麼，移轉後的那個人，是真的延續了原來的連續性，還是只是一個承載了你所有數據參數的新生兒？

如果從「確保本尊唯一性」的資料一致性角度來看，意識移轉是一個跨節點的分散式交易 (Distributed Transaction)。為了避免產生「兩個你」或是「你死在傳輸過程中」，系統必須保證絕對的原子特性。「意識移轉」在類似於 分散式交易與二階段提交 (Two-Phase Commit, 2PC) 的概念。

在雲端應用裡很常見的虛擬機飄移技術，像是 VMware vMotion 或 Proxmox Live Migration 技術，讓一台虛擬機從實體機 A 飄移到實體機 B 的過程，這個過程虛擬機內部的運作是無感的，這個也是類似意識移轉的概念。

在影集《碳變》中，意識被儲存於皮層盤中隨意插拔替換肉體；電影《阿凡達》則透過神經連結艙，讓意識訊號無縫接入異種生物軀體，在另一個現實中尋找新的認同；漫畫《七龍珠》那美剋星篇，悟空與基紐特戰隊隊長交換身份的段落，就是一種意識移轉。

不過「移轉」過程如果需要有「複製」動作，如果沒有刪除原本 A 的意識，其實代表就有可能造成下一個問題 ...

▌誰才是本尊的悖論：意識複製 (Clone)​

「移轉」過程有個「複製」，如果這個「複製意識」沒有被刪除，那麼就會產生另一個「自我」。當然，我是假設「意識」是可以透過技術手段做到複製，目前無法被證實，所以只能說是個思想實驗。而意識可以被複製，背後衍生的意涵則是記憶也同樣被複製了。因為記憶是一個人格的體現，意識則需要根據記憶形成人格。這概念跟 LLM 需要一堆資料訓練，資料就是記憶的一種。

意識可以複製、記憶隨之可以複製、備份，放到載體後同時啟動，A 與 B 同時存在且擁有完全相同的記憶數據時，「獨一無二的我」就崩解了。

遊戲《SOMA》將「擲硬幣」的生存悖論發揮到極致，主角透過掃描將意識數據複製到新機器人身上後，原本的自己只能被遺留在黑暗的深海中；而在《Cyberpunk 2077》裡，強尼銀手的意識被複製成「Relic」晶片中的數位印跡，結局中「靈魂殺手 (Soulkiller)」的設定與網路空間的描繪，更是缸中之腦最殘酷的數位化變體。

我之前寫的一段短篇故事 返老還童，就是在討論複製意識衍生的道德與識別問題。

▌只有意識，沒有記憶？​

上一段我提到複製意識，也會跟著複製記憶的假設。但如果意識與記憶是分開的呢？如果只有複製意識，卻沒有記憶，那這個「自我」是否就回到「原廠設定」，也就是剛出生的嬰兒，一個完全不認識任何人，沒有過往的記憶，但是具備生物本能，需要重新學習控制手腳。

那如果把這個意識放到其他生物，例如小強、貓、狗，這大概就是民間所謂的「投胎」？

第三章：沒有意識的空殼（硬體複製）​

▌出廠預設值的悲哀：肉體 / DNA 複製​

與意識複製相反，如果我們只複製了相同的軀體，但沒有寫入任何意識數據，這具生化空殼將會透過新的感官輸入，發展出自己獨立的意識。這探討的是「先天與後天的辯證」以及「被造物的存在價值」。
《銀翼殺手》(仿生人夢見電子羊) 中的複製人雖然擁有與人類無異的肉體，甚至被預載了假記憶，但他們在接收世界訊號後所運算出的情感，往往比真實人類更加純粹；《新世紀福音戰士》中的綾波零，則是一具具等待靈魂注入的複製容器，探討了自我價值的虛無與尋找。

結語：在成為神之前，我們還剩下多少人性？​

回到我一開始提到的核心問題：「『我』究竟是什麼？」

當我試圖用工程理性與矩陣思維，將人類、機械與意識無情地拆解為「硬體載體」與「軟體數據」時，我們看似理清了科技演進的脈絡，但其實也同時打開了潘朵拉的盒子。這份排列組合的清單，原本應該只存在於《駭客任務》或《Cyberpunk 2077》的虛構世界裡；然而，隨著 Neuralink 腦機接口的商用化、Tesla Optimus 等實體機器人的量產，以及 AI 運算能力的爆發式進化，這些科幻設定正一步步從「思想實驗」變成不可逆的「工程現實」。

我們必須保持極度警惕：當科技賦予我們隨意替換零件、甚至抽換意識的能力時，我們失去的可能不只是肉體。如果意識真的能被完美複製與移轉，當「你」在另一個機械軀殼中甦醒時，那個留在原地的「本尊」，是被延續了，還是在資料覆寫的瞬間被悄悄抹殺了？如果記憶可以像硬碟資料一樣被輕易讀寫與竄改，我們該如何確認此刻的悲喜不是一場演算法植入的幻覺？

正因為在不遠的未來，記憶可能淪為一串串可以被備份、修改甚至偽造的電子訊號，我們才更應該在此刻，重新審視「真實回憶」的無可取代性。 那些真正定義「我們是誰」的，從來不是大腦裡冰冷完美的資料庫，而是我們與家人、朋友、愛人共同度過的那些充滿生物性瑕疵、卻又無比真實的瞬間——一頓平凡熱騰騰的晚餐、一次沒有行程的漫步、一場相擁而泣的告別。請好好珍惜這些與身邊親友相處的時光，用心去創造更多專屬於你們的共同回憶。因為這些伴隨著真實體溫、氣味與情感交織而成的羈絆，是任何超級電腦都無法演算，也是浸泡在營養液裡的「缸中之腦」永遠無法真正觸碰的珍寶。

科技發展的速度，永遠比道德倫理的防線崩潰得更快。當我們急於褪去這具脆弱的生物皮囊，追求機械的強大與數據的永恆時，或許我們該停下來問問自己：當所有的硬體都被替換，所有的五感訊號都能被偽造，最後留下的那個東西，還能稱之為「人類」嗎？

探索這些排列組合，不應只是為了對未來的科技奇觀發出驚嘆，它更應該是我們在全面墜入「缸中之腦」的深淵之前，為自己敲響的最後一記警世鐘聲。

2026.03.03 at Taipei
