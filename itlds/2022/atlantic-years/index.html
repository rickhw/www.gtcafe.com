Atlantic Years

X-Japan
90 年代充滿傳奇色彩的日本樂團
#視覺系 始祖
傳奇的 鼓手 / 鋼琴 / 作曲 / 製作人 - Yoshiki (林佳樹)
1998 年自縊 的 lead guitarist: hide

回到正題
照片的這張 CD
標題是: Singles
加 S 的 single 是啥鬼 .... XD
總之
是 1993 年出版的 X-Singles 系列專輯
其中的第二張
副標題是: Atlantic Years




這張專輯是我第一張擁有 X 的 CD
這張也是 "海歸" 的
是學生時代
一個朋友從日本帶回來送我的
在有這張 CD 之前
其實還沒真的聽過 X 原版的歌曲
什麼意思？
那是個網路剛開始的年代
那年代要取的音樂不容易
mp3 這種東西也剛起來
大部分的電腦是無法聽 mp3 (記得要 P100 以上)
學生也買不起 CD .... (紅標?
而那是我剛開始在摸索編曲的年代
MIDI (SMF) 檔案格式也因為檔案小
在網路上流傳
所以我大概有幾年的時間
聽的音樂都是純 MIDI
配上我那時候第一次面交
買的 DB50XG 音源子卡 (要價 3000 大洋)
所以 X 的歌這樣聽了幾年
才有機會聽到原曲

回到這張 CD
很特別的是
一張 CD 裡面居然曲名叫做 Tears 的有三軌
根本是騙錢 XDD
三軌版本分別是: Studio, Classic, Live 三個
其他收錄的單曲都很有名
X 迷都很熟悉
Rusty Nail, Week End,
Forever Love, Crucify My Love, Longing

以前還流行 "#床頭音響" 的年代
(這東西應該已經退出世界舞台了 QQ
學生時代因為特殊原因
(其實是理由 有機會再說 XD
我在某一年的中秋節
花了大把存的錢買了一台 JVC 音響
還記得是可以同時放 3 張 CD
兩張 錄音帶
可以定時錄音 播放
可以處理 AUX 音源 (我都當電腦音響用)
在那年代可是炫砲 + 奢侈的設備

有好長的時間
這張 CD 是每天早上
透過床頭音響自動播放的一張
每天早上既是以淚(誤) 醒來 XDD

Youtube 找不到現成的 playlist
不過專輯的歌
每首都是 X 的名曲
滿滿回憶的一張 CD
