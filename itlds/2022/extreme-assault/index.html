Extreme Assault

我會玩的某個遊戲或者看某部電影
有很大的因素就是音樂
甚至是因為音樂而玩遊戲的
這個遊戲就是一個例子



照片是學生時代買的遊戲
是那年代 (1997?) 少數的 PC 3D 遊戲
這個遊戲是個戰爭模擬遊戲
玩家可以駕駛直昇機或者坦克攻城掠地
大概是這樣

這個遊戲光碟除了資料片
本身也有 CD 音軌
音軌就是遊戲的 OST
不同於一般的遊戲使用 MIDI / MOD 透過 CPU 即時運算
他在遊戲中是直接播 CD 音樂
讓玩遊戲的沈浸感更深刻 (至少在那年代)
(另外一個 有 CD 音軌的是仙劍奇俠傳, 但遊戲中還是以 MIDI 為主)
而這個音樂的製作正式讓我直接買的主因

還記得是跟一個社團的學弟去逛建國商場
店員在解說這遊戲的時候
我聽到音樂整個就黏住了
根本不管遊戲在玩啥鬼
3D 有多魔炫砲 (那年代 FF7 剛出來沒多久)
在店員示範玩幾個 run 之後
直接掏錢就買了
沒記錯的話
大概是 900 大洋

就這樣這個在當年
少數的 3D 戰爭遊戲
被我買回家當 CD 聽
那時候寫 Code 的時候
都聽這張 "CD" ....... XDDD

不誇張
每一首曲子
都很有 "氣氛"
經過廿多年
現在聽起來都還是熱血沸騰

Spotify 常常會有調查軟體工程師寫扣都聽啥音樂
大部分的清單怎麼看都很普通
不是一些金屬
smooth jazz
沙發音樂 還是電音
就是一些芭樂
(沒有別的意思 ... XDD)

然後
一定有人會問
那我到底有沒有玩過這遊戲？
其實有
因為買的時候
店員給我了一組金鑰
只要輸入就會變無敵
所以我玩過幾次
(五指手指頭數得出來XD)
但是大部分的時間
還是聽音樂居多

這年代很幸福
不需要去啥地下 FTP 找 mp3
反正有關鍵字上 youtube 大概都會有
連 playlist 都準備得好好的 XDD
這張我真的很推薦
工作時
要進入 #flow 狀態的人
都可以聽聽看
旋律的 hook 很強
很有 "前進感"
也很有危機感 (戰爭, schedule)

遊戲玩起來長啥樣？
鄉民都錄好了
其實現在看起來還是滿好玩的 (？

#過年就是回憶時刻
#記憶是最珍貴的禮物
