工作中的樂團

紀錄一段故事

差不多是去年的這段時間 (2021/10 - 12)
同事找我幫忙要準備尾牙表演
年紀大了
這種體力活已經沒啥興趣
不過可以去練團室走走倒是可以玩玩看
然後就不小心掉進一個坑 XDD

動機​

其實就是讓尾牙更熱鬧
或者弄一些新花招

對我來講
我只是想去練團室彈彈琴
拉底賽之類

不過最後就是我在幻想
因為現實往往不是這樣的

確立編制​

附圖表是當時我寫的三個 Plan
依照人員編制，以及需要的器材設備
由複雜 (Full Set) 到簡單 (部分預錄)

預算考量，
有限的預算無法滿足 Full Set 所需要的
像是內場 PA 設備租賃、外場音響
最後以實際可以跑的經驗，
直接網 Plan C (部分) 走
一來減少表演成員 (8 -> 5)
再來就是簡化設備問題

實際上我上面寫的這段文字
大概就是我自己一個人在自嗨
因為能了解 想像的人不多

不過預算考量
從這種方式來溝通
最後還是把方向定下來了

成員確立：樂手、歌手​

方向確立了，
接下來就是把人就定位
分成樂手和歌手
歌手透過選拔，找出前三名

樂手就比較麻煩
因為大部分的同仁不是長期在玩音樂
所以別說彈一首歌，
可能連自己的設備都搞不清楚有什麼
像是吉他手沒有自己的效果器
導線有問題
Bassist 可能 N 年沒彈了

總之大家都很有熱情
不管怎樣
先把人確立下來
技術與設備
就走一步算一步了 XD
我心想，最糟就是全部預錄
上去演演戲就好 XDD

確立歌單​

樂手, 歌手都確立的
接下來就歌單

活動時間約 15m
去頭去尾，實際表演的時間約 10-12m
保守一點抓 10m
歌大概就是準備四首, 實際上去可能是 3 首

歌單以歌手擅長的為主
所以讓歌手列
這概念大概就是讓業務隨便開需求差不多
最後一定開出各式各樣風格的歌曲

歌曲也要符合老中青三代都聽過
也要符合尾牙活動主題.
但實際執行上來講
樂團組成就會限制歌的表演的路線了
所以最後選出幾首
已經就被樂手組成限制了

好不容易選出來了
曲目丟給主辦單位
卻說要跟特定主題有關才行
這個決策會讓所有人炸鍋

最後忘了怎麼談的
反正維持原案的歌單
來來去去 兩週又過去了

歌單總算確立了
接下來才真的要開始

總譜​

有樂手, 有歌手, 有歌單
練團室租好了
要去練團了

去練團室就開始練？
音樂是時間藝術
所有人必須遵循基本的框架
所謂的練習才有意義

確立歌單後
我就週末把三首歌的第一版總譜寫完
上班之前就給大家

請大家做幾件事情，首先是樂手:

  1. 確立總譜看得懂
  2. 確立總譜各個段落正確的
  3. 確立大家對和弦沒問題

再來是歌手：

  1. 請歌手確立歌曲原調是否唱起來覺得舒服？
  2. 雙主唱的部分請歌手自己先討論怎麼安排段洛
  3. 歌手有三位，沒有唱的人，可以去唱和聲

大家在我的「淫威」之下
很快就進入狀況
幫忙 debug 總譜
也了解接下來要準備做的任務了


預錄音軌：鼓、合成器、鋼琴、弦樂、音效​

除了總譜，
一開始提到 Plan C 只有兩把吉他和 Bass
所以原曲其他以外的東西都是預錄的
也就是我先在家編好
帶去現場播放
主要是鼓還有其他鍵盤的部分 (弦樂, 合成器, 鋼琴)

因為一開始就跟大家對總譜
所以 Backing 製作就是依照總譜跑
只要有人跑拍
跑錯段落 整個就會 GG
因為 Backing 不像真人可以自動應變
(Backing 就是更簡易版的 Kala)

另外 Backing 跟總譜
我都是每個 Milestone 不斷迭代修正

靈敏的人看到這裡會發現
三首歌單確認的那一週
隔週要練團前
我要準備三首歌的總譜
以及三首 Backing 製作
還好，以前練團時，
這種事都是我在做

另外，Backing 一開始我也只是快速拉出鼓的部分
讓大家可以對段落與拍子
接下來真的是迭代式改進
附圖照片是我當時在編曲時的實況

開始練團​

練團的目的，就是要確保表演順利
樂手、歌手能夠順利演出。
前面說了，大概掌握所有人狀況後
#刻意練習 這必然的。
要讓大家花時間練習的方法
就是製造時間的迫切感，
直接用專案管理很基本的: #以終為始

直接把大約 10weeks (2.5m) 的時間拉出來
把所有練團時間 book 下來
練團室訂下來
設定幾個 milestones.
直接預告下週我就會把總譜給大家
樂手記譜、歌手記歌詞 ... 等

然後，就開工了！
整個 time line 大概是這樣：

  • M1 (2w): 樂手把譜記下來、歌手熟記歌詞
  • M2 (2w): 歌手, 樂手 開始不能看譜, 看詞
  • M3 (2w): 修歌曲段落，符合表演目標的總長度 (10m) ，扣除進出場
  • M4 (2w): 在練團室，模擬設備狀況 / 關燈練習、樂手歌手走位
  • M5( 2w): 每次都是同正式演出, 服裝, 燈光

這段大概是最開心的過程了
雖然有夠累了
每天上班已經夠累了
一把老骨頭
下班還要扛著超重的琴
去練團室再撐三小時
那時候又天天下雨
又濕又冷又餓又累又想睡覺

不過 Backing 一下，
和弦刷下去就是反射動作的彈。

附上的錄影，
應該是在 M2 or M3 階段的練習.

計劃趕不上變化​

尾牙的前一週，
台灣疫情升溫
我們依舊到練團室練習
已經開始越來越多公司宣布取消尾牙

2022/01/13 星期四
公司宣佈尾牙活動形式從飯點的方式
改到公司內部 叫外賣的方式
簡單說：取消 .......

原本準備的活動呢？
改成線上直播 ....
.......
現場直播 + 串流
這個坑很深
我踩過不少雷
太多技術問題要克服
短短一週是不可能的
所以改成全部預錄
所以要有畫面 + 聲音

畫面​

本來現場活動就打算錄畫面
所以我們就到公司攝影棚
拍了幾個 take
放練團室的錄音
大家抓了一些 take
然後聲音怎麼辦？

聲音：主唱錄音​

畫面路好，隔天星期五
下週就尾牙
所以星期五請同仁去訂一間練團室
同仁支援麥克風

星期五晚上下班後
所有人到練團室
幫忙錄主唱的部分

之前我做的 Backing
除了前面提到的鼓 / 弦樂 / 鋼琴 之外
檔案裡也有我自己練習的錄音
剛好是一個半成品的 kala

就這樣用半成品的 Kala
然後一群人去練團室
收主唱的音

三首歌，每首歌收 3 個 take
三位主唱都很優秀
one take, one take, one take

不到晚上九點
三首歌全部收音完畢

其中有一首有和聲
現場我靈機一動
找所有人一起唱和聲
作爲後製混音的素材

收完所有聲音的部分
接下來才是進後製

聲音：吉他錄音​

前面提到，星期五晚上錄的是主唱和和聲
我有提到 Kala 其實是半成品
隔天星期六我醒來一想

cow ... 我要把剩下的吉他都彈完
還有 Bass .... 囧

所以週六一天，我一個人就彈了三首歌
兩把吉他的所有部分，包含 Rhythm + Solo
然後還有 Bass .....
一天搞定三首歌的錄音
都快發瘋了XDDD

混音：一天三首​

星期六搞定吉他錄音
接下來
星期天就到混音了
這不是我擅長的部分
但是也只能在一天之內把三首歌都混完

這段就挑過去不寫了
因為太雷了 XDD
大概跟一天要把三個服務 CI / CD 搞定一樣

影像​

我在處理聲音的同時
視覺完全由大家處理
所以拍出很搞笑的 MV
最後聲音和影像疊再一起
才是大家看到的東西

真的是演一演啊​

一開始就想:

最糟就是全部預錄, 大家上去演演戲, 開心就好

沒想到變成真的
只是除了歌手之外的東西
全部都是我彈的
而且還是在最後一週的週末 QQ

雖然是坑
但現在回味起來
還真的很有趣
跟群非常優秀
且熱血的年輕朋友
在練團室嘶吼
這就是青春啊 XDD
